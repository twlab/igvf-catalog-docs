# Field Lineage

This page documents how fields flow from **portal files** (with [file format specifications](https://data.igvf.org)) through [KG loading adapters](https://github.com/IGVF-DACC/igvf-catalog/tree/dev/data/adapters) into **ArangoDB** and the **REST API**.

For automated deep traces, use the `trace` tool in [igvf-catalog-dev-agents](https://github.com/IGVF/igvf-catalog-dev-agents).
