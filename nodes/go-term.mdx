## GO Term Annotation

This table provides detailed information about Gene Ontology (GO) term annotations associated with biological entities.

| Column | Description |
|--------|-------------|
| Symbol | Gene product symbol |
| Name | Name of the annotation (click for annotation details) |
| GO ID | Unique identifier for the GO term |
| GO Term Name | The specific Gene Ontology term name |
| Type | Type of gene product (e.g., protein, RNA) |
| Qualifier | Additional qualifiers for the GO term annotation |
| Evidence | Code indicating the type of evidence for this annotation |
| Organism | The organism to which this annotation applies |
| Source | Origin of the GO term information |
